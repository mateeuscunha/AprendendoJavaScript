	Condição com else:

> if(condição){
> bloco de código
> }
> else{
> bloco de código
> }

	Condição com else if:

> if(condição){
> bloco de código
> }
> else if(condição) {
> bloco de código
> }
> else if(condição) {
> bloco de código
> } 
> else { 
> bloco de código
> }
